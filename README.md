# 🎥 TikTok Style Video Player

A responsive TikTok-style vertical video feed built using React.
This project replicates core reel functionalities like auto-play, scrolling, likes, and comments.

## ✨ Features

* 🎬 Vertical scroll video feed
* ▶️ Auto play/pause using IntersectionObserver
* ❤️ Like functionality (click + double tap)
* 💬 Comment modal
* 🔊 Mute / Unmute
* 📖 Read more (expand caption)
* 🔁 Infinite scrolling
* 📱 Responsive UI

---

## 🛠 Tech Stack

* React (Vite)
* JavaScript
* CSS

---

## ⚙️ Setup Instructions

```bash
npm install
npm run dev
```

---

## 🧠 Approach

* Used IntersectionObserver for active video detection
* Ensured only one video plays at a time
* Built reusable components for UI
* Focused on smooth scrolling & UX

---

## ⚠️ Limitations

* Static video data (no backend)
* Some videos are landscape (handled with responsive fit)
